 На основе алгоритма Прима придумайте жадный алгоритм решения задачи коммивояжера. 
Протестируйте алгоритм на случайных данных, которые представляют собой точки на евклидовой плоскости, полученные в результате случайного выбора N центров на плоскости согласно нормальному распределению с sigma = S1 вокруг точки (0,0) и генерации вокруг каждого из этих центров M точек согласно нормальному распределению с sigma = S2.
Рассмотрите, например, случай S1=100, S2=10, N=100, M=100. Сравните алгоритм с алгоритмом ближайшего соседа по среднему значению найденного пути. Экспериментально найдите долю случаев, когда один алгоритм даёт результат лучше другого.

Для получения нормального распределения можно воспользоваться преобразованием Бокса-Мюллера.
